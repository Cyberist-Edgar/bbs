# bbs

> A website build with vue , also see [origin one](https://github.com/Cyberist-Edgar/2020-Software-Engineering) without vue

### How to run
- install the packages
    ```
    npm install  
    ```

- run the project
    ```
    npm run serve
    ```

技术栈：
- `vue`  主要的js开发框架
- `nodejs`  开发环境
- `vue-router`  路由处理
- `vue-cli3`  脚手架
- `vuex`  状态管理
- `mockjs`  数据模拟
- `axios`  数据请求
- `antdv`  UI组件库
